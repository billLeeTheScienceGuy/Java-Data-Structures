# Interactive GUI Simulation: Graphical State Management

## Overview
An interactive, event-driven graphical application built using Java and the Processing core library. This simulation demonstrates object-oriented design principles by rendering and managing an array of dynamic graphical objects within a visual window.

## Technical Concepts Demonstrated
* **Event-Driven Programming:** Implemented active event listeners (`mousePressed`, `keyPressed`) to trigger asynchronous state changes based on user input.
* **Graphical Object State Management:** Designed dynamic hit-box collision detection to allow users to click, drag, and interact with specific objects rendered on the screen.
* **External Library Integration:** Utilized a third-party graphics library (`processing.core`) to handle the continuous rendering loop, managing visual assets and background environments.
