# Static Array Manipulation: Climbing Tracker

## Overview
A Java-based utility designed to track and calculate statistics for bouldering/rock climbing sessions. This project demonstrates core data manipulation using static arrays, string parsing, and algorithm design for basic statistical analysis.

## Technical Concepts Demonstrated
* **Static Array Management:** Safely appending data to fixed-size arrays and maintaining state variables for active capacity.
* **Data Validation & Parsing:** Implementing logic to parse alphanumeric string inputs (e.g., "V3", "V5"), validate formatting, and convert them into numerical data for mathematical operations.
* **Algorithm Design:** * Calculating moving averages based on dynamic historical subsets.
  * Generating visual, string-based histograms to map the frequency distributions of successes and failures across different difficulty grades.
